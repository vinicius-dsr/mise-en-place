# Mise en Place

Mise en Place é uma agência de marketing gastronômico premium, focada em transformar restaurantes em destinos gastronômicos de sucesso. Nós ajudamos a elevar sua marca e atrair clientes por meio de estratégias de alto impacto.

## Tecnologias Utilizadas

Este projeto foi construído utilizando as seguintes tecnologias:

- **React** (via Vite)
- **TypeScript**
- **Tailwind CSS** para estilização rápida e responsiva
- **Lucide React** para ícones modernos

## Como Executar o Projeto

1. Instale as dependências:
   ```bash
   npm install
   ```

2. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

3. Para gerar uma versão de produção:
   ```bash
   npm run build
   ```
